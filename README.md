# Uppgift 7 från lab 3 i Algoritmer och datastrukturer, fast i C
## Clonea repon
```git
git clone https://github.com/Phillezi/HI1029_LAB3_Uppg7_C.git
```
```bash
cd HI1029_LAB3_Uppg7_C
```
## Kompilera
### Linux/MacOS
```bash
make
```
### Windows med MINGW
```powershell
mingw32-make.exe
```
## Kör
```bash
./main
```
